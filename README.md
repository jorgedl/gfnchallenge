# Geofusion UI

#### Antes de qualquer coisa, execute:

`npm install`

#### Servidor de desenvolvimento local

`npm start`<br />
Acesse http://localhost:8005/

#### Build para produção

`npm run build` <br />
Todo o conteúdo para produção será gerado no diretório /dist

#### Tests

`npm run test` <br />
Testes unitários serão executados no terminal
